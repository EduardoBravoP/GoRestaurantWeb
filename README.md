## How to run:
```bash
# Clone this repository
$ git clone https://github.com/EduardoBravoP/GoRestaurantWeb.git

# Go into the repository
$ cd GoRestaurantWeb

# Install dependencies
$ yarn

# Run JSON server
$ yarn json-server server.json -p 3333

# Run the app (Web)
$ yarn start
```
